# テスト仕様書

最終更新: 2025-02-08

## 1. テスト方針

### 1.1 テスト種別

| 種別 | 対象 | ツール |
|------|------|--------|
| 単体テスト | ユーティリティ関数、バリデーション | Jest |
| APIテスト | APIルート | Jest + fetch mock |
| E2Eテスト | 主要ユーザーフロー | 手動 / 将来Playwright |
| 手動テスト | UI・画面遷移・決済 | ブラウザ |

### 1.2 テスト環境

- DB: Supabase（開発環境）
- 決済: Stripe テストモード
- メール: Resend テストモード
